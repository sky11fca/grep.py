USAGE: python3 grep.py <regex_pattern> <path/to/object> [OPTIONS]

OPTIONS:

-ignorecase    Lists all lines that match with the pattern ignoring case.
-not           Lists all lines that do not match with the pattern.
-count         Counts the number of lines that match with the pattern. If it is a directory, it computes the number of lines for each file plus total matches

REGEX PATTERN WILDCARDS:

'.'             Every character except newline.
'^'             Beginning of the line.
'*'             Repeats zero or more times.
'?'             Repeats zero or one time.
'+'             Repeats one or more times.
'{int}'         Matches exactly int occurrences.
'{int, int}'    Matches int to int occurrences.
'[]'            Group of characters.
'|'             Or operator.
'\d'            Decimal digit.
'\D'            Anything except a digit.
'\s'            Whitespace character.
'\S'            Anything except whitespace.
'\w'            Alphanumeric character.
'\W'            Anything except alphanumeric.
'\'             Escape character.
'[^...]'        Group of characters not in [...].
'()'            Grouping.
'[...-...]'     Intervals of characters.
